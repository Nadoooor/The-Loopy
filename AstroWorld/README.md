# AstroWorld
A VR game about planets where you are in space and you can explore each planet and know information about the sun, earth, mars, etc.

----------------
### How to play:
1. open the deploied link for this project from [here](https://nadoooor.github.io/AstroWorld/)
3. see the controls down there.
2. and that's it enjoy ;)

-------------------
### Controls:
- wasd to move
- shift to sprint
- T to call the Control-Panal
- Click with mouse or touch to interact with buttons or interact with planets to get Info cards.

-------------------------

### How to Build:
1. Clone the Repository
2. edit what ever you want
3. use the live server extension to deploy your changes locally 
4. enjoy and please star :star:
